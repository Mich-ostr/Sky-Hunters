# 🎮 Sky Hunters: Ascenso Infinito

## 📌 Descripción
Videojuego en Canvas 2D donde el jugador elimina enemigos que ascienden.

## 🎯 Objetivo
Aplicar animaciones, eventos del mouse y lógica de videojuegos.

## ⚙️ Funcionalidades
- Movimiento aleatorio
- Eliminación con click
- Cambio de color al hover
- Niveles progresivos
- Score y porcentaje

## 🎮 Controles
Mouse = apuntar  
Click = eliminar
